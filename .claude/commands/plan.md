1. Lis la demande attentivement

4. Découpe en étapes numérotées (max 5-7 étapes)
5. Pour chaque étape, liste les fichiers touchés

```
## Plan : [fonctionnalité]

### Étapes
1. [action] → fichiers: [liste]
2. [action] → fichiers: [liste]
...

### Vérification
- [ ] Build passe
- [ ] Lint passe
- [ ] Pas de régression sur l'existant
```
